gPhoto + RPi2 + Emberjs

Welcome to snappyapp where you can create swarms of cameras and control them individually and all at once.

Official site

The stack

  1. resin.io - For provisioning and managing the device fleet
  2. pubnub - For the pub/sub and command triggering
  3. firebase - For the data layer
  4. auth0 - For user identity managment of the swarm admin
  5. emberjs - For the swarm admin app

Hardware Setup

  1. Raspberry Pi 2 Model B
  2. Your camera of choice. Supported gphoto2
  3. For remote triggering
